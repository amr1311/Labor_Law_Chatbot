# Labor_Law_Chatbot
 Labor Law Chatbot is an AI system that answers questions about employment laws automatically. It uses a retrieval system (RAG) to fetch relevant legal information and a language model to generate accurate responses. The goal is to provide fast, reliable legal guidance without needing to consult a lawyer for basic questions.

 
🧠 # How It Works

1.The user asks a question

2.The system retrieves relevant legal documents (ChromaDB)

3.The retrieved context is sent to the LLM

4.The model generates a grounded answer based on the context

🛠️ Tech Stack

Python

Streamlit

ChromaDB
